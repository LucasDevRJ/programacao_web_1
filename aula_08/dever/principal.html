A programação back-end é uma das áreas fundamentais do desenvolvimento de software, responsável por criar e gerenciar a lógica que acontece "por trás dos bastidores" de sistemas e aplicações. Diferente do front-end, que lida com a interface e a interação do usuário, o back-end cuida do processamento de dados, comunicação com bancos de dados e integração com serviços externos.

As linguagens mais comuns para programação back-end incluem Java, Python, PHP, Node.js (JavaScript), Ruby, entre outras. Além das linguagens, o back-end envolve o uso de frameworks e bibliotecas que facilitam o desenvolvimento, como Spring para Java, Django para Python e Express para Node.js.

Os profissionais que atuam com back-end também precisam ter conhecimentos sobre bancos de dados (relacionais, como MySQL, ou não-relacionais, como MongoDB), segurança da informação, APIs, autenticação de usuários, e práticas como versionamento de código e deploy em servidores.
